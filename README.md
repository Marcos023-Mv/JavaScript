# javaScript

📚 Atividades de JavaScript
Este repositório reúne atividades práticas desenvolvidas durante os estudos de JavaScript, cobrindo os principais conceitos fundamentais da linguagem. As atividades foram criadas e organizadas utilizando o Visual Studio Code como editor, explorando desde os primeiros passos até recursos mais interativos da linguagem.
Ao longo das atividades foram trabalhados temas como a exibição de mensagens com o clássico Olá Mundo, a declaração e uso de variáveis e seus diferentes tipos, a aplicação de operadores aritméticos, lógicos e de comparação, além da captura de dados por meio de entradas do usuário. Também foram praticadas estruturas condicionais para tomada de decisão e estruturas de repetição para automatizar tarefas. A organização do código foi desenvolvida com o uso de procedimentos e funções, separando responsabilidades e tornando o código mais reutilizável.
Na parte de interatividade, foram explorados os principais eventos do JavaScript, incluindo eventos de mouse, eventos de teclado e o evento input, que permite reagir em tempo real às ações do usuário na página.
🛠️ Tecnologias Utilizadas

HTML5
CSS3
JavaScript
Visual Studio Code

🚀 Como Executar
Clone o repositório, abra a pasta no Visual Studio Code e execute o arquivo .html da atividade desejada diretamente no navegador ou utilizando a extensão Live Server.
